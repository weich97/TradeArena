# Evidence Label Checklist

Use only labels that are supported by the artifact:

- `stress-only`
- `cached-provider`
- `live-provider`
- `deterministic-baseline`
- `external-submitted`
- `quote-calibrated`
- `fill-replay-validated`
- `redacted-prompt`
- `fully-auditable`

Model skill claims require repeated runs, baselines, statistical uncertainty,
and provider/version provenance.
