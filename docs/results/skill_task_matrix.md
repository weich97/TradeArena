# TradeArena Skill Task Matrix

This file is generated by `python scripts/score_skill_task_report.py`.
The suite evaluates LLMs and coding agents as financial-audit agents,
not as stock pickers or live trading assistants.

TradeArena skills help agents inspect, reproduce, audit, and extend benchmark artifacts.
They are not trading strategies, investment advice, broker integrations, or live execution tools.
If a benchmarked model uses a skill, the skill must be recorded as part of the model, prompt, or retrieval configuration.
